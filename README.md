# 🌐 Portfolio Website

A personal portfolio website built from scratch to showcase my skills, projects, and progress as a MERN Stack Developer.

## 🚀 Purpose

This project was created to practice building a modern, responsive, and interactive website using core frontend technologies.

## 🛠️ Tech Stack

- HTML5 - Semantic structure
- CSS3 - Responsive design, Flexbox, Grid, animations
- JavaScript - Interactivity and DOM manipulation

## ✨ Features

- Responsive navigation
- Modern portfolio layout
- Hero, About, Skills, Projects, and Contact sections
- Dark / Light mode toggle
- Typing animation
- Mobile-friendly design
- Hover effects and animations
