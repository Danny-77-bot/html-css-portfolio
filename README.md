# Portfolio Website Project

A personal portfolio website built from scratch to practice and showcase my foundational front-end development skills. 

## 🚀 Project Purpose
The goal of this project is to master core web technologies by building a clean, modern, and responsive user interface. This serves as a landing page to feature my future projects, skills, and contact information.

## 🛠️ Tech Stack
*   **HTML5** - Semantic markup for structure and accessibility.
*   **CSS3** - Custom styling, layout design (Flexbox/Grid), and media queries for responsiveness.

## 📈 Key Features Practiced
*   Creating a responsive navigation bar.
*   Structuring clean, semantic sections (Hero, About, Projects, Contact).
*   Implementing a mobile-first or desktop-first responsive design.
*   Styling interactive buttons and hover states.
